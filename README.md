# Info-X
Info-X is a information gathering tool...with this tool You can collect vectims information like ip , gmail , phone number information....

### Features :
+ Track IP Address
+ Phone Number Info
+ E-Mail Info

### Tested on:
+ Termux
+ Linux
### Available for:
+ Termux
+ Linux

+ ```apt upgrade```

+ ```git clone https://github.com/TeamTSZ/Info-X.git```

+ ```python Info-X.py```
